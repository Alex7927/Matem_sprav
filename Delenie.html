Деление
Деление — это математическое действие, обратное умножению, с помощью которого по произведению и одному из множителей находят другой множитель.
a : b = c

Компоненты деления:

  • a — Делимое (число, которое делят)
  • b — Делитель (число, на которое делят)
  • c — Частное (результат деления)
⚠️ ЗАПОМНИ: На ноль делить нельзя!

Связь с умножением:
Чтобы проверить результат деления, нужно частное умножить на делитель: c × b = a
