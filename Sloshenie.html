Сложение
Сложение — это математическое действие, при котором два или несколько чисел объединяются в одно целое.
A + B = C

Компоненты сложения:

  • A — Первое слагаемое
  • B — Второе слагаемое
  • C — Сумма
Главное свойство (Переместительное):
От перемены мест слагаемых сумма не меняется.
a + b = b + a
