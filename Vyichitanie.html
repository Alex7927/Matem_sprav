Вычитание
Вычитание — это математическое действие, с помощью которого от одного числа отнимают другое число.
a - b = c

Компоненты вычитания:

  • a — Уменьшаемое (число, которое уменьшают)
  • b — Вычитаемое (число, которое вычитают)
  • c — Разность (результат действия)
Как проверить вычитание?
Нужно к разности прибавить вычитаемое. Если получится уменьшаемое — пример решен верно!
c + b = a

Важное свойство:
Если из числа вычесть то же самое число, получится ноль: a - a = 0.
Если из числа вычесть ноль, число не изменится: a - 0 = a.
